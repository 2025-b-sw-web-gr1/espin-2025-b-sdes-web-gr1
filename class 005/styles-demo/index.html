Comparación: TAG vs CLASS vs ID vs STYLE (inline)

Reglas aplicadas (orden esperado de menor a mayor prioridad): TAG < CLASS < ID < STYLE (inline). Una clase con !important puede cambiar esto.

1) Selector por TAG (p): este color viene de la regla global para el tag p.

2) Selector por CLASS (.example-class): esta regla proviene del archivo CSS (clase).

3) Selector por ID (#example-id): el ID tiene mayor especificidad que la clase.

4) STYLE (inline): tiene la máxima especificidad entre estos ejemplos y gana sobre ID y CLASS.

5) Clase con !important: puede sobrescribir un ID sin !important.
