Formas de cargar estilos en una página web

Este proyecto muestra varias técnicas: estilo en línea, <style> interno, hoja externa, @import, y carga dinámica vía JavaScript.

1) Estilo en atributo (inline)

Este párrafo usa el atributo style="...".

2) Estilo interno (<style> en el documento)

Ejemplo con reglas definidas dentro de la página en un bloque <style>.

3) Hoja externa (<link rel="stylesheet" href="...">)

Esta regla viene de styles/external.css (y ese archivo hace @import de otro).

4) @import dentro de una hoja CSS

El archivo styles/external.css contiene un @import "imported.css". Es otra forma de encadenar estilos.

5) Carga dinámica desde JavaScript

Prueba los botones para cargar estilos dinámicamente (crea elementos <link> o <style>).

6) Ejemplo: evitar bloqueo del render

Usamos rel="preload" as="style" onload="this.rel='stylesheet'" arriba para mostrar una técnica de carga no bloqueante y fallback en <noscript>.
