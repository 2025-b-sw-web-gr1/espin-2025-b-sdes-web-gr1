# JSONPlaceholder API - Swagger


## Objetivo
Documentar y visualizar endpoints REST de la API JSONPlaceholder utilizando OpenAPI 3.0 y Swagger UI con fines educativos.


## Tecnologías utilizadas
- Swagger Editor
- Swagger UI
- OpenAPI 3.0
- JSONPlaceholder (API pública)


## Estructura del proyecto


jsonplaceholder-swagger/
├── docs/
│ └── openapi.yaml
├── swagger-ui/
│ └── index.html
└── README.md


## Uso


1. Abrir `docs/openapi.yaml` en VS Code
2. Copiar su contenido en https://editor.swagger.io/
3. Visualizar y probar los endpoints desde la interfaz
4. (Opcional) Abrir `swagger-ui/examen.html` con Live Server


## Endpoints documentados
- GET /posts
- GET /comments
- GET /users


## Notas
Este proyecto es únicamente para fines académicos y de aprendizaje.
