# Proyecto: Demostración de Formas de Cargar Estilos CSS

Este proyecto educativo muestra las diferentes formas de cargar estilos en una página web:

1. **Etiqueta `<style>` en el `<head>`**
2. **Archivo CSS externo**
3. **Atributo `style` en línea**
4. **Estilos aplicados dinámicamente con JavaScript**
5. **Estilos cargados desde un servidor externo (CDN/HTTP)**

## Archivos
- `index.html`: Página principal con ejemplos y explicaciones.
- `styles-demo.css`: Archivo CSS externo.

## Cómo ejecutar el proyecto

1. Instala Node.js si no lo tienes: https://nodejs.org/
2. Instala http-server globalmente:
   ```
   npm install -g http-server
   ```
3. Abre una terminal en la carpeta `styles-demo` y ejecuta:
   ```
   http-server
   ```
4. Abre tu navegador y visita la URL que aparece (por ejemplo, http://localhost:8080)

¡Explora los diferentes ejemplos y aprende cómo se cargan los estilos en la web!
