# API de Equipos y Jugadores

Este proyecto documenta una API RESTful con relación 1 a muchos,
donde un equipo de fútbol puede tener múltiples jugadores.

## Tecnologías usadas
- Bruno (cliente HTTP)
- OpenAPI / Swagger (documentación)

## Entidades
- Teams
- Players

## Relación
- GET /teams/{id}/players
